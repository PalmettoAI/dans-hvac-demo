## Cold-Call Angles

### 1. "You're losing jobs to voicemail"
Small HVAC owner-operators miss calls constantly when they're on the job. The demo shows a web form that captures name, phone, and service type — so leads don't evaporate when they can't answer.

### 2. "Your competitors are showing up first — you're not"
Without a meta description or schema markup, they're invisible in local search. The demo is already SEO-tagged with `HVACBusiness` schema — tell them Google can now read exactly what they do and where.

### 3. "You built this business on referrals — let's make it look the part online"
Owner-operator pride is real. They've earned their reputation; the site makes it visible. When someone Googles them after a referral, what do they see right now? Probably nothing impressive.

### 4. Seasonal urgency (spring/summer)
Hit them before AC season hits. Homeowners start calling in March–April when temperatures spike. A site that ranks and converts is worth the most right now.
